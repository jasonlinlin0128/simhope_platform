# 管理員後台操作指南

> 適用對象：持有 `admin` 帳號的管理員（目前為 IT / 經企室負責人）

---

## 登入

1. 開啟工具箱首頁，點右上角 **「👨‍💻 開發者登入」**
2. 輸入 admin 帳號（email）和密碼
3. 登入後 Navbar 右側出現 **「🛠️ 管理後台」** → 點擊進入 `/admin`

---

## 後台三大功能 Tab

### 1. 🧰 工具管理

列出所有已提交的工具（含待審核、開發中、已上線等所有狀態）。

**審核 / 改狀態**

每個工具右側有一個下拉選單，可直接切換狀態：

| 狀態 | 何時使用 |
|------|---------|
| `使用中 live` | 驗收通過，正式開放同仁使用 |
| `測試中 beta` | 功能完整但需要試用期 |
| `新上線 new` | 剛上架，特別標示 |
| `開發中 dev` | 仍在開發，暫不顯示給一般同仁 |
| `待驗收 pending` | 開發者提交後的預設狀態 |
| `已終止 terminated` | 停止維護 |

**刪除工具**

每個工具右側有 **「🗑️ 刪除」** 按鈕，點擊後會出現確認視窗，確認後永久刪除 Firestore 文件。

**編輯工具說明書**

點擊工具名稱旁的 **「✏️ 編輯」** → 進入工具詳情頁 → 點頁面右上角 **「✏️ 進入編輯」** 可編輯 block 內容（即使不是原作者，admin 也有權限）。

---

### 2. 😤 痛點卡片管理

列出所有已提交的痛點卡片（before / after 格式）。目前支援查看，尚無 inline 編輯。

---

### 3. 👥 帳號管理

#### 建立開發者帳號

1. 在「帳號管理」Tab 下方，填入：
   - **電子郵件**（需為 `@simhope.com.tw` 或指定網域）
   - **密碼**（至少 8 碼）
   - **顯示名稱**（選填，預設用 email 前綴）
2. 點 **「＋ 建立帳號」**
3. 系統使用 secondary Firebase App 建立，**不會登出管理員**

#### 修改帳號角色

每個帳號右側有角色下拉選單，可在 `developer` / `admin` 之間切換。

> 注意：刪除帳號目前需透過 Firebase Console，後台尚未支援。

---

## 注意事項

- **「🏠 首頁設定」** 按鈕目前為施工中，點擊無效。
- 狀態變更即時生效（寫入 Firestore），同仁重新整理頁面即可看到最新結果。
- 若更新失敗，頁面會顯示「更新失敗，請重新整理後再試」，重整後重試。
