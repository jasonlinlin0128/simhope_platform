🚀 怎麼取得這個資源

這個工具有 3 種使用方式，依照你的需求挑一個就好。

⭐ 推薦給大多數人

📦 一鍵安裝包 (.mcpb)

適合誰：用 Claude Desktop 的同仁，不想碰指令列

下載一個檔案 → 雙擊 → Claude 就會問你要不要啟用。3 分鐘搞定。

⬇️ 下載 iso-form-mcp.mcpb
給技術同仁

🧰 npm 指令安裝

適合誰：用 Cursor / VSCode、會編輯 config 檔的人

把下面這段貼到你的 MCP config 裡：

{ "mcpServers": { "iso-form": { "command": "npx", "args": ["-y", "@simhope/iso-form-mcp"] } } }
不會技術也能用

📄 直接下載 ISO 表單 ZIP

適合誰：不想接 AI，只想拿原始 PDF/Word 表單

包含 28 份 ISO 表單原始檔，定期更新版本。

⬇️ iso-forms-v2026.05.zip

deliveryMethods 的 kind 列舉

kind 使用者看到的標籤 主要欄位
webapp🌐 網頁應用url
download⬇️ 軟體下載downloadUrl, platform (win/mac)
doc📄 文件/表單downloadUrl, fileType (pdf/docx/xlsx/zip)
mcp🔌 AI 連接器mcpbUrl, npmPackage, repoUrl, configSnippet
api🧩 API / SDKendpoint, docsUrl, sdkPackage
showcase📺 僅展示（無 — 純說明）

遷移策略

  1. 新欄位 deliveryMethods 跟舊欄位 type+url 並存（向下相容）
  2. 讀取時：優先用 deliveryMethods，若不存在則 fallback 把 type+url 轉成單一 deliveryMethod 物件
  3. 寫一個 migration script，把現有 14 筆 tools 自動補上 deliveryMethods（從 type+url 推導）
  4. 新提交強制走 deliveryMethods，admin/dashboard 表單改版
